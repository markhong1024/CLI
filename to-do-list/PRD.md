# PRD: 오늘의 할일 관리 웹페이지

## 1. 개요 (Overview)

오늘 해야 할 일을 추가, 완료, 삭제할 수 있는 심플한 To-Do List 웹 애플리케이션.
별도 서버 없이 브라우저만으로 동작하며, 새로고침 후에도 데이터가 유지된다.

---

## 2. 목표 (Goals)

- 오늘의 할일을 빠르게 추가하고 관리할 수 있다
- 완료한 항목을 시각적으로 구분할 수 있다
- 페이지를 새로고침해도 데이터가 사라지지 않는다
- 별도 설치 없이 HTML 파일 하나로 실행된다

---

## 3. 사용자 (Target User)

- 개인 일정을 간단히 관리하고 싶은 누구나
- 복잡한 앱 없이 브라우저 하나로 to-do를 관리하고 싶은 사람

---

## 4. 핵심 기능 (Core Features)

### 4-1. 할일 추가
- 텍스트 입력 후 Enter 또는 "추가" 버튼 클릭으로 항목 생성
- 빈 문자열은 추가 불가 (유효성 검사)

### 4-2. 완료 처리
- 항목 클릭 또는 체크박스로 완료/미완료 토글
- 완료된 항목은 취소선 + 흐린 색상으로 표시

### 4-3. 삭제
- 각 항목 우측의 삭제 버튼으로 개별 삭제
- 완료된 항목 전체 삭제 버튼 제공

### 4-4. 진행 현황 표시
- 전체 항목 수 / 완료 항목 수 표시 (예: 3 / 7 완료)
- 진행률 프로그레스 바 표시

### 4-5. 데이터 영속성
- localStorage를 사용해 브라우저에 데이터 저장
- 새로고침 후에도 목록 유지

---

## 5. 비기능 요구사항 (Non-functional Requirements)

| 항목 | 내용 |
|------|------|
| 기술 스택 | HTML5, CSS3, Vanilla JavaScript |
| 외부 의존성 | 없음 (CDN, 라이브러리 사용 안 함) |
| 저장소 | localStorage (서버 불필요) |
| 반응형 | 모바일/데스크탑 모두 지원 |
| 파일 구성 | 단일 index.html 또는 html + css + js 분리 |

## 7. 개발 범위 (Scope)

### In Scope
- 할일 추가 / 완료 토글 / 삭제
- localStorage 저장
- 진행률 표시
- 반응형 레이아웃

### Out of Scope (추후 고려)
- 날짜별 필터링
- 우선순위 설정
- 알림 기능
- 서버 동기화 / 로그인
