# PRD — 종스크롤 슈팅 게임 (Xevious-style)

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **프로젝트명** | Game01 |
| **장르** | 종스크롤 슈팅 (Vertical Scrolling Shooter) |
| **레퍼런스** | 제비우스 (Xevious, 1982) |
| **기술 스택** | HTML5 Canvas, CSS3, Vanilla JavaScript (ES6+) |
| **플랫폼** | 웹 브라우저 (PC 기준) |
| **작성일** | 2026-06-15 |

---

## 2. 핵심 목표

- 브라우저에서 즉시 실행 가능한 단일 파일 또는 폴더 구조의 웹 게임
- 제비우스 스타일의 위→아래 배경 스크롤과 적 등장 패턴 구현
- 공중 적(Air Enemy) + 지상 적(Ground Enemy) 이중 공격 시스템
- 60 FPS 기준 부드러운 렌더링 (requestAnimationFrame)

---

## 3. 파일 구조 (예정)

```
game01/
├── prd.md          ← 이 파일
├── index.html      ← 게임 진입점
├── style.css       ← UI / 화면 레이아웃

```

## 4. 기능 요구사항

### 4-1. 화면 구성

| 요소 | 설명 |
|------|------|
| 캔버스 크기 | 480 × 640 px (세로형) |
| 배경 스크롤 | 지형 타일이 위→아래로 무한 반복 스크롤 |
| HUD | 상단: 점수(SCORE) / 하단: 남은 생명, 폭탄 수 |
| 화면 상태 | 타이틀 → 게임 중 → 게임오버 → (재시작) |

### 4-2. 플레이어

| 항목 | 내용 |
|------|------|
| 이동 | 방향키 또는 WASD — 상하좌우 자유 이동 |
| 공중 샷 | Space — 전방 기관포 (Zapper) 연사 |
| 지상 폭탄 | Z 키 — 아래쪽 폭발탄 (Blaster) 1발 발사 |
| 무적 시간 | 피격 후 2초간 깜빡임 |
| 생명 | 시작 3기 |

### 4-3. 적 — 공중 (Air Enemy)

| 종류 | 행동 패턴 | 점수 |
|------|----------|------|
| 일반 전투기 | 직선 하강 or 곡선 이동 | 100 |
| 편대 전투기 | 4~8기 V자 / 一자 편대 비행 | 200 |
| 보스기 | 화면 상단 등장, 다방향 탄막 | 5000 |

### 4-4. 적 — 지상 (Ground Enemy)

| 종류 | 행동 패턴 | 점수 |
|------|----------|------|
| 고정 포대 | 플레이어 방향으로 포탄 발사 | 300 |
| 이동 포대 | 좌우 이동하며 포탄 발사 | 500 |

### 4-5. 스테이지 & 스크롤

- 스테이지 1개 기준 구현 (루프 가능)
- 스크롤 속도: 기본 2 px/frame, 보스 구간 1.2 px/frame
- 웨이브 스폰 데이터를 배열로 정의 (스크롤 Y 위치 기반 트리거)

### 4-6. 충돌 판정

- AABB (축 정렬 경계 박스) 방식
- 공중 레이어 / 지상 레이어 분리 — 지상 적은 공중 샷에 반응 안 함

### 4-7. 파워업 (선택 구현)

| 아이템 | 효과 |
|--------|------|
| 쉴드 | 1회 피격 무효 |
| 샷 강화 | 2-way / 3-way 탄 |
| 폭탄 추가 | 폭탄 +2 |

## 5. 비기능 요구사항

| 항목 | 기준 |
|------|------|
| 렌더링 | 60 FPS 목표 (requestAnimationFrame) |
| 호환성 | Chrome / Edge 최신 버전 |
| 외부 의존성 | 없음 (라이브러리 미사용) |
| 에셋 | Canvas 2D API로 직접 드로잉 (이미지 파일 없이 구현) |
